# goedit
A text editor built using golang

[![forthebadge](https://forthebadge.com/images/badges/made-with-go.svg)](https://forthebadge.com)

## Supports:
- macOS
- windows
- linux

## Building:
```
$ git clone https://github.com/krishpranav/goedit
$ go mod tidy
$ ./build.sh
```
